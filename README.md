# EMNIST-HOG-SVM_LOOCV_Muhammad-Faisal3969
This is a simple dataset test program for demonstrating EMNIST letter classification using HOG and SVM with LOOCV as the evaluator. Here you will learn about the usage of EMNIST dataset as the sample and the implementation of vision using HOG and SVM with LOOCV to learn how it's function as classifier and evaluator.
